# Bubble Hit Game – JavaScript Mini Game

A fast-paced, interactive browser game built using **HTML, CSS, and Vanilla JavaScript**.  
Common!!! Test your speed, focus, and reflexes by clicking the correct number before time runs out!

This project focuses on **game logic, UI/UX design, micro-animations, and clean DOM manipulation**.

## Features

-  **Dynamic Target System** – A random number is generated every round
- **Countdown Timer** – Limited time to score maximum points
- **Score Logic** – Score increases only on correct hits
- **Modern UI Design** – Gradient backgrounds and smooth layout
- **Event Delegation** for better performance
- **Responsive Layout** (works on most screen sizes)
